Haalt tot 300 keer per dag de laatste weergegevens op van het KNMI.
Om nieuwe data op te halen moet de actie flow uitgevoerd worden.

Het antwoord kan afgevangen worden met de trigger kaart, welke vele variabelen geeft om in je flow te gebruiken.
Om deze app te kunnen gebruiken is maar 1 variabele nodig, een api key van weerlive.nl welke gratis aan te vragen is.
